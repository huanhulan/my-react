# My-React

My simple implementation of **Functional Component Only** `React`.

## Supported React API

- [x] Functional Component
- [x] render()
- [x] cloneElement()
- [x] createContext()
- [x] createElement()
- [x] createPortal()
- [x] createRef()
- [x] hooks()
- [x] memo()

## Mono-repo structure

```
.
└── packages
    ├── app
    └── my-react
```

## Playground

Install dependencies

```
yarn
```

Then run the following command

```
yarn start
```

## Build

Run the following command

```
yarn build
```
